# Cobertura de decisão para a classe ExportComparator

O método escolhido compara as duas strings **s1** e **s2** e retorna 0 se for igual

|ID| Condição | Situação para True | Situação para False |
| :-: | :-: | :-: | :-: |
| 1 | s1 == s2 | s1 = s2 | s1 != s2| 
| 2 | s1 != s2 | s1 != s2 | s1 = s2|
